# **Workout Tracker App**

A web app to track your workouts, view them by month, and manage exercises easily.
Built with React for the frontend and FastAPI for the backend.

## **Features**

Add, edit, and delete workouts
View workouts grouped by month for easy navigation
Workout detail view with all exercises for that session
Dynamic updates: UI updates automatically after adding or deleting a workout
Responsive design: works on desktop and mobile
Simple and intuitive UI for easy use

## **Tech Stack**

Frontend: React, Axios
Backend: FastAPI
Database: SQLite (or your choice)
State Management: React useState and useEffect

## **Future Improvements**

Add statistics and charts (monthly progress, total lifts)
Add presets
Add user adding exercises to the database
